**Data Types in C** <br>

There are _4_ Data Types in C: <br>
1. Basic Data Type: _int, char, float, double_ <br>
2. Derived Data Type: _array, pointer, structure, union_ <br>
3. Enumeration Data Type: _enum_ <br>
4. Voice Data Type: _void_
